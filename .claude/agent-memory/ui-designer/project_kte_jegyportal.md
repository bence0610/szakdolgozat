KTE Jegyportál is a Hungarian football ticketing platform for Kecskeméti TE. All UI text is in Hungarian. The product is a web app (Angular frontend, NestJS backend).

**Why:** Szakdolgozat (thesis) project. Design and implementation must be thorough and professional.

**How to apply:** Treat every UI design request as production-grade. Hungarian copy throughout.

## Epics that explicitly require UI design (marked in backlog)

- E2 – Landing Page & Public UI (KTE-008 to KTE-012)
- E3 – 2.5D Isometric Stadium Map (KTE-013 to KTE-019)
- E5 – Authentication & User Profile (KTE-028, KTE-029)
- E8 – Loyalty Points System — loyalty dashboard (KTE-045)
- E10 – Admin Dashboard (KTE-061, KTE-062)

## Epics that require UI but are NOT explicitly flagged in backlog

- E4 – Cart Summary Page (KTE-023) — inferred from acceptance criteria
- E6 – Checkout Page & Order Confirmation (KTE-032, KTE-036) — inferred
